<h2>Čo je Movies4u?</h2>
<p>Movies4u je aplikácia ktorá slúži ako databáza filmov a seriálov</p>
<p>K jednotlivým filmom a seriálom priradí linky na rôzne stránky kde si môže používateľ zhliadnuť filmy</p>
